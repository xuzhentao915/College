大学生向けに設計されたモダンなキャンパスアシスタント。最新の **Skyline** レンダリングエンジンを搭載。

## 🌟 主な機能

### 1. インテリジェントなリマインダー
*   **試験カウントダウン**: 残り日数を自動計算するグラスモーフィズムカード。残り3日を切ると**赤色**に変化します。
*   **お知らせスワイパー**: キャンパスニュースをスワイプで確認。重要度に応じたタグ付け機能を搭載。

### 2. モダンデザイン (Visuals)
*   **グラスモーフィズム**: すりガラス効果と拡散シャドウを採用し、高級感のあるレイヤー構造を実現。
*   **リニアアイコン**: 3px の統一された線画アイコンによる一貫した視覚体験。

### 3. Skyline 最適化
*   **スクロール体験**: Skyline エンジン専用の `scroll-view` レイアウトにより、滑らかな操作感を実現。
*   **ナビゲーション調整**: iOS/Android のノッチに対応した独自コンポーネントを搭載。
